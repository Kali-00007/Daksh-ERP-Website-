# Daksh ERP Website

GitHub Pages ready static website.

## Upload Rules
Upload the CONTENTS of this folder to the repository root:
- index.html
- assets/
- data/
- lang/
- docs/
- downloads/
- updates/

Do not upload the parent folder itself.

## Enable GitHub Pages
Repository Settings -> Pages -> Deploy from branch -> main -> root.

## Editable Content
- data/site-config.json
- data/plans.json
- data/features.json
- data/downloads.json
- data/releases.json
- data/faq.json
- lang/en.json
- lang/hi.json
- updates/version.json
